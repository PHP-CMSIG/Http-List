# Http List

## Installation

Add the package via [composer](https://getcomposer.org) to your PHP project:

```bash
composer require php-cmsig/http-list
```

## Usage

### Query List

```php
use PhpCmsig\HttpList\Cookies;

Cookies::getCookies();
```

### Query List

```php
use PhpCmsig\HttpList\Queries;

Queries::getQueries();
```
